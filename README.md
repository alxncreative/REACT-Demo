# DEMO for React Dev Course

This is a simple TODO list application built with React and Vite.

## Requirements

You must have Node.js installed (version 18 or newer recommended).

You can check with:

```
node -v
```

## Installation

1. Clone or download the project.

2. Open a terminal in the project folder.

3. Install dependencies: `npm install`

## Development
1. Run the app `npm run dev`
2. Then open the address shown in the terminal (usually): `http://localhost:5173`

This mode reloads automatically when it detects changes on files.

## Run the production build
1. Build the optimized production version: `npm run build`
2. Then preview it locally: `npm run preview`
3. Open the address shown in the terminal (usually): `http://localhost:4173`

## Notes
The built app cannot be opened directly from the dist folder using file://.
It must be served through a local server, which is what npm run preview does.
